Framework Overview

PR3TACK organises tactics and techniques and maps them to preemptive countermeasures. The Matrix is searchable and community-extensible. Techniques are triaged by feasibility and impact.

How to use

  1. Browse tactics and techniques in the matrix.
  2. Filter by platform, feasibility, or tags.
  3. Export a JSON seed or submit a contribution.

Examples

Atom Table Abuse: Windows Global Atom table used for cross-process payload retrieval. Captured as Persistence & Defense Evasion.

Revix Kill-All-VMs: Cloud resilience erosion via mass VM termination; captured under Resilience Erosion.

Case Studies

Atom Table Abuse

Windows Atom Tables can be abused to store small payloads for cross-process retrieval. Although demonstrated in research, it was not immediately captured in ATT&CK.

Revix Ransomware: Kill-All-VMs

A destructive tactic terminating virtual machines at scale, causing resilience loss. PR3TACK maps this to Resilience Erosion and suggests guard rails and immutable snapshots as defences.
