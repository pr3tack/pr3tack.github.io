Anticipate threats before exploitation

PR3TACK catalogues plausible-but-unobserved TTPs and maps them to preemptive countermeasures for defenders, vendors, and policy makers.

Execution

BIOS Config Trigger Payloads
Low — Enforce BIOS password & Secure Boot; audit BIOS config changes; firmware inventory and signing.
Peripheral Firmware Stager
Medium — Vendor firmware verification, restrict peripheral provisioning, allowlist peripheral vendors, attestation for USB devices.

Persistence

Abuse of Windows Atom Tables
High — Monitor GlobalAddAtom/GetAtomName API usage, EDR hooks for Atom APIs, limit unnecessary Global Atom usage.
Hidden Service Registration
Medium — Service creation monitoring, image signing enforcement, service account least-privilege, periodic service inventory.

Privilege Escalation

Exploit Virtual Device Drivers (VDDs)
Medium — Restrict loading of unsigned drivers; enable Device Guard; kernel-mode code signing enforcement; driver vetting.
Credential Store Forging
Medium — Credential hygiene, short token lifetimes, monitor unusual credential store writes, token binding where possible.

Defense Evasion

Cloud Metadata API Overloading
Medium — Limit access to metadata endpoints, metadata proxying with auth, audit API calls for anomalies, token scope minimization.
Telemetry Tampering
Medium — Agent integrity checks, signed agent updates, out-of-band telemetry integrity verification, multi-path telemetry.

Command & Control

Bluetooth HID Covert Channels
Low — Disable unnecessary Bluetooth services; monitor HID traffic; endpoint controls for wireless interfaces.
Covert DNS-over-HTTPS Tunnels
Medium — DoH proxy allow-listing, recursive resolver controls, inspect DoH at enterprise proxies, behavior-based detection.

Discovery

Browser Extension Enumeration via IPC Abuse
Medium — Limit extension API privileges; monitor inter-process communications; browser hardening policies.
Identity Topology Mapping
Medium — Identity flow hardening, federation metadata vetting, privileged identity mapping alerts, least-trust segmentation.

Collection

Abuse of Clipboard History Services
High — Monitor clipboard access frequency; enforce clipboard isolation in sensitive apps; disable system-wide clipboard history.
Credential Cache Scraping via Accessibility APIs
Medium — Harden accessibility API privilege model, require signed accessibility helpers, monitor accessibility API calls.

Exfiltration

Leveraging Screen Reader APIs for Data Leak
Medium — Enforce signed driver use for accessibility APIs, limit privileged access to screen buffers, screen-content redaction.
Covert Cloud Storage Staging
Medium — Cloud storage access controls, DLP on uploads, object tagging & provenance checks.

Lateral Movement

Software Update Supply Chain Piggybacking (Unpopular Apps)
Medium — Validate software update servers, scan binaries for tampering, allowlisting, supply-chain attestations.
Business-Unit Policy Propagation
Medium — Strict RBAC, entitlement reviews, cross-unit change approvals, admin role segmentation.

Pre-Positioning

OSS Supply-Chain Sleeper Commit
Medium — Code review gating, provenance checks, CI/static analysis, vet third-party dependencies.
Anticipatory Brand/Domain Squatting
High — Brand monitoring, defensive registrations, DMARC/DKIM/SPF enforcement, legal watch.

Resilience Erosion

Backup Bit-Rot Injection
Medium — Immutable backups, frequent restore drills, end-to-end checksums, out-of-band backup integrity logs.
Patch-Fatigue Campaigns
Medium — Vulnerability triage, vendor-proof verification, centralised patch governance.

Governance Subversion

Standards/Comments Manipulation
Low — Security/legal representation on standards bodies, review of proposals, internal policy audits.
Procurement Account Establishment
Medium — Supplier verification, contractual security clauses, onboarding attestations, procurement checks.

Cognitive Manipulation

Analyst Habituation Lures
High — Adversarial training, alert hygiene, purple-team injects.
Adversarial Log Crafting
Medium — Log-schema validation, sanity checks, rotation of detection logic, threat-hunting heuristics.

Operational Tempo Manipulation

Holiday/Travel Window Triggering
High — Follow-the-sun coverage, escalation backups, redundancy, travel blackout policies.
Billing/Cost Shock Forcing
Medium — Billing anomaly detection, budget guardrails, immutable infra, throttling policies.

AI/ML Subversion

Training-Data Seed Poisoning
Medium — Data provenance tracking, dataset signing, validation, adversarial-ML red-team tests.
Model Output Injection
Medium — Model monitoring, output validation, dual-model cross-checks, rejection thresholds.

Sociotechnical Pressure

Investor/Media Narrative Orchestration
Low — Comms + security sync, fact-check pipeline, exec training, pre-approved response plans.
Vendor-Influence Campaigns
Medium — Procurement security checklist, acceptance gates, exec sign-off for major adoptions.

Digital Exhaust Manipulation

IOC/Telco Feed Pollution
High — Source reputation scoring, feed vetting, automated IOC quality checks, human review for critical IOCs.
Telemetry Noise Flooding
Medium — Signal-to-noise monitoring, sampling strategies, baselining, telemetry integrity tests.
